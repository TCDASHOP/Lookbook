# SAIREN COLOR ARCHIVE — Lookbook (Full Replace Package)

## Structure
- `/index.html` : Archive landing (links to issues)
- `/issue-01/` : Magazine-style issue (cover → 20 looks → back cover)
- `/assets/img/looks/` : Put your look images here (filenames must match exactly)
- `/robots.txt` + `/sitemap.xml` : basic SEO plumbing
- `/404.html` : GitHub Pages friendly 404

## Images (絶対にリネームしない)
次のフォルダに、各LOOKの **webp と jpg を両方** 入れてください：
`/assets/img/looks/`

例：
- assets/img/looks/007_hs_mt_men.webp
- assets/img/looks/007_hs_mt_men.jpg

## Deploy
GitHub Pages の公開ルートにこのまま差し替え（rootに index.html が必要）。
